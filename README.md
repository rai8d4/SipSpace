# WebDev2FinalProject
#SipSpace

## Project Summary

SipSpace is a personal cafe journal where you are able to log every coffee shop visit with structured data including what drink you got, your rating of said drink, 
productivity level, work log, and general ambiance of the environment.
Over time, usage of this app allows you to keep a personal database of your unique cafe life!

 A lot of students, remote workers, and creatives use cafes as “third spaces” to get work done or just feel more focused. 
 SipSpace works as a personal log of these spaces, so users can go back and see which cafes they liked best and where they actually got things done.
